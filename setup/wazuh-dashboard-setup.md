# Wazuh Dashboard Setup

The Wazuh Dashboard provides a web-based interface for monitoring and analyzing security events.

## Installation Overview
- Installed on the Wazuh VM as part of the SIEM stack
- Configured to connect to the Wazuh Manager and Indexer

## Access
- Accessed via web browser using the dashboard URL
- Used for alert monitoring and investigation

## Verification
- Dashboard loads successfully
- Security alerts and logs are visible

## SOC Relevance
Dashboards allow SOC analysts to quickly identify, investigate, and respond to security events.
