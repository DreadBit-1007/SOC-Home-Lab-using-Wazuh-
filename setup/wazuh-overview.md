# Wazuh Architecture Overview

Wazuh is used as the SIEM platform in this SOC home lab to collect, analyze, and visualize security events.

## Core Components
- **Wazuh Indexer:** Stores and indexes security event data
- **Wazuh Manager:** Processes logs and generates alerts
- **Wazuh Dashboard:** Provides visualization and analysis interface
- **Wazuh Agent:** Collects logs from monitored endpoints

## Purpose
This architecture simulates how a SOC monitors and investigates security events using a SIEM platform.
